# Launch Instructions

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Commands

In the directory /backend run:

### `node server.js`

This starts the backend server.\\

Now in another terminal window, go to the project directory and run:

### `npm start`

This runs the app and should automatically open it in your browser.\
Otherwise you can open [http://localhost:3000](http://localhost:3000) to view it in the browser.
